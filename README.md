# Open Financial Data Protocol (OFDP)

OFDP is a consumer-owned, federated, open standard for personal financial data.

## Overview

OFDP defines how financial data is structured, stored, shared, and verified across institutions, applications, and AI systems. It is designed to complement FDX and Section 1033 infrastructure by adding a consumer data sovereignty layer that institutions, aggregators, and AI agents can consume natively.
